About this extension

This plugin (extension) is for chrome-based browsers with extension support. This extension adding some abilities in your Pony Town game. For now, this plugin have only one feature so this plugin have the name "Pony Importer Reborn". Why "reborn", this is because this plugin is based on Pony Importer Chrome extension that is stopped to be developed and supported so I'm going to give the new life of this extension for all of you guys. I'm also be able in my free time to add more features to this plugin so everyone will be able to better modify their Pony Town game expirence. I'm giving the full support to this extension and I'm ready to hear your sugestions if you have some and add your sugestions to be the feature of this plugin. Be aware that I will rename this plugin and website when I add some features in this plugin so, stay tuned!
